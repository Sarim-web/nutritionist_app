NutriTrack, also known as VitaCalo, is a comprehensive, offline-first calorie and nutrition tracking mobile application built with Flutter, designed to empower users with personalized nutrition guidance while prioritizing privacy and accessibility. Developed for cross-platform deployment on Android, iOS, web, Windows, Linux, and macOS, the app focuses on helping individuals monitor their dietary intake without relying on internet connectivity or external servers, ensuring data remains entirely local and secure.

At its core, NutriTrack serves as a privacy-focused nutrition companion that allows users to log meals, track calorie consumption, and build healthier eating habits. The app's offline architecture uses Hive, a lightweight NoSQL database, for local data storage, eliminating the need for cloud accounts or data sharing. This approach protects user privacy, as all information—ranging from food logs to personal profiles—stays on the device. Users can export data as CSV files or share reports via the integrated sharing functionality, providing flexibility for personal analysis or consultations with nutritionists.

Key features include a robust onboarding process where users complete a detailed survey to establish their profile, including dietary preferences, goals (e.g., weight loss, maintenance, or muscle gain), activity levels, and health metrics. This personalized setup enables the app to offer tailored recommendations. The dashboard provides a real-time overview of daily, weekly, and monthly calorie intake, macronutrient breakdowns (proteins, carbs, fats), and progress toward goals, visualized through interactive charts powered by the fl_chart library.

Calorie logging is central to the app, supporting multiple input methods: manual entry from a local food database, voice input for hands-free logging, and camera integration via image_picker for photographing meals (though automatic recognition is not implemented, photos aid in manual tracking). The food database, stored in a JSON file, includes over 50 global and regional items (e.g., Pakistani dishes like Chicken Biryani or Nihari, Indian options like Paneer Tikka, and universal foods like apples or grilled chicken), categorized by meal type (breakfast, lunch, dinner, snacks) and region. Users can add custom foods, and the app calculates nutritional values based on portion sizes.

Advanced features encompass diet building, where users create meal plans aligned with their profiles; historical logs for reviewing past intake; and graphical analytics for trends in calorie consumption and weight changes. The profile management system allows switching between multiple user profiles (e.g., family members), each with isolated data. Settings offer customization for themes (light/dark mode), units (metric/imperial), language preferences, and data management options like clearing logs or importing/exporting data.

Localization is a standout feature, supporting multiple languages through Flutter's internationalization framework, with ARB files for translations. The app includes connectivity checks to alert users of offline status and permission handling for camera and storage access. Additional utilities include a welcome screen for first-time users and a survey system for ongoing profile updates.

Technologically, NutriTrack is built on the Flutter SDK and Dart programming language, ensuring native performance across platforms. It employs state management via the Provider package (with optional Riverpod integration), routing with Go Router, and local storage through Hive Flutter. Data handling includes CSV for export/import, Path Provider for file system access, and Share Plus for sharing capabilities. Image Picker enables photo capture, while FL Chart provides data visualization. Connectivity Plus and Permission Handler manage network status and device permissions. The app adheres to Material Design principles, uses Flutter Localizations and Intl for i18n, and incorporates Flutter Launcher Icons and Native Splash for branding. Concepts like offline-first architecture, NoSQL databases, modular feature-based code organization, privacy-by-design, and cross-platform compatibility drive its development.

In summary, NutriTrack is an all-in-one nutrition tool that combines ease of use, comprehensive tracking, and strong privacy guarantees, making it ideal for individuals seeking sustainable dietary management without compromising data security. Whether for personal health goals or professional guidance, it provides the tools needed for informed, offline nutrition tracking.</content>
